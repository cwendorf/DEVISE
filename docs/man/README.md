# [`DEVISE`](https://github.com/cwendorf/DEVISE/)

## Reference

This section is the reference for `DEVISE`. Each page documents one exported function, including usage, arguments, return values, and examples.

### Setup

- [create_groups()](./create_groups.md): Create grouping vectors.
- [name_rows()](./name_rows.md): Set row names of a data frame.
- [name_columns()](./name_columns.md): Set column names of a data frame.

### Selection

- [use_vars()](./use_vars.md): Select variables or formulas from a data frame.
- [filter_rows()](./filter_rows.md): Filter rows referenced by a formula.
- [keep_as()](./retain.md): Keep selected values, rows, or columns in a pipeline.
- [extract_rows()](./extract_rows.md): Extract one or more rows from a data frame.
- [extract_columns()](./extract_columns.md): Extract one or more columns from a data frame.
- [extract_vector()](./extract_vector.md): Extract a named row or column from a data frame.
- [extract_intervals()](./extract_intervals.md): Extract point estimate and interval bounds from output.

### Computation

- [compute_descriptives()](./compute_descriptives.md): Compute descriptive statistics.
- [compute_correlations()](./compute_correlations.md): Compute correlation or covariance matrices.
- [compute_metrics()](./compute_metrics.md): Compute additional metrics for statistical estimates.

### Output

- [style_matrix()](./style_matrix.md): Style and print a formatted matrix.
- [plot_conditions()](./plot_conditions.md): Plot point and interval estimates by condition.
- [plot_comparison()](./plot_comparison.md): Plot comparisons confidence intervals.

# [`DEVISE`](https://github.com/cwendorf/DEVISE/)

## Reference

This section is the reference for `DEVISE`. Each page documents one exported function, including usage, arguments, return values, and examples.

### Setup

- [create_groups()](./create_groups.md): Create grouping vectors for conditions or designs.
- [name_rows()](./name_rows.md): Assign row names.
- [name_columns()](./name_columns.md): Assign column names.

### Selection

- [retain()](./retain.md): Keep selected values, rows, or columns.
- [use_vars()](./use_vars.md): Select variables for downstream analysis.
- [extract_rows()](./extract_rows.md): Extract one or more rows.
- [extract_columns()](./extract_columns.md): Extract one or more columns.
- [extract_vector()](./extract_vector.md): Extract a named row or column.
- [extract_intervals()](./extract_intervals.md): Extract interval bounds and related fields.

### Computation

- [compute_descriptives()](./compute_descriptives.md): Compute descriptive statistics.
- [compute_correlations()](./compute_correlations.md): Compute correlation statistics.
- [compute_metrics()](./compute_metrics.md): Compute standardized metrics used in reporting.

### Output

- [style_matrix()](./style_matrix.md): Format matrices or tables for publication-ready output.
- [plot_conditions()](./plot_conditions.md): Plot estimates by condition.
- [plot_comparison()](./plot_comparison.md): Plot comparisons and associated intervals.
